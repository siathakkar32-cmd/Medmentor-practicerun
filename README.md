# MedMentor

MedMentor is an AI-powered student guidance platform designed to help high school students explore healthcare careers, track extracurriculars, find opportunities, plan courses, and build stronger college applications.

## Features

- Student profile builder
- Readiness dashboard
- AI mentor chat
- Career explorer
- Opportunities finder
- Clinic finder
- College analyzer
- Extracurricular tracker
- Course planner
- Resume builder
- Resource library

## Project Status

This is the first working web prototype of MedMentor. Future versions may include user accounts, a database, login, personalized saved profiles, and a mobile app.
